My First Project in PyQt5. 
A simple Text Editor which have these functionalities:

1. Making a new Txt File
2. Openig a existing Txt File
3. Saving Txt File
4. Basic operations: Undo/Redo and Cut/Copy/Paste - added by coding, instead of using predefined functions
5. Exit function. 
6. Save on Exit
7. Help Menu 

Modifications will be done in this Text Editor to add more functioanlities like Text Formatting, Find and Replae. etc..
